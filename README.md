# === Анализ банковских операций ===

Приложение для анализа финансовых операций: загружает данные из Excel-файла, фильтрует и группирует их, а также строит отчёты и сохраняет их в лог-файлы.

## Возможности

-  Загрузка и обработка Excel-файлов и JSON-файлов с транзакциями
-  Простой поиск по ключевым словам в описании или категории
-  Формирование отчётов по категориям трат
-  Сохранение и обработка логов
-  Тестирование функций ( ~ 90% )
-  Структура проекта с использованием `Poetry`, `pytest`, `pandas`

---

## Cтарт

1. **Запуск приложения:**
```
poetry run python main.py
```

2. **Запуск тестов:**
```
poetry run pytest --cov
```
